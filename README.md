## 📌 简介 
The skill-code repository is designed to store code for various Agent skill modules. This project is primarily used for the development, maintenance, and management of automated skill scripts across different business scenarios. <br>
**code-skill** 是一个用于存放各类 Agent 技能模块代码 的仓库。该项目主要用于开发、维护和管理不同业务场景下的自动化技能脚本。
